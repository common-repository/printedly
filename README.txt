An easy to use online designer for designing custom T-shirts with a quick quote tool. Free to use.

== Description ==

Printedly Online Designer is an easy to use platform to provide your customers with an online designer and quick quote tool to customize products such as T-shirts, Mugs and Caps on a your WordPress website.

= Unlimited Products, Orders and Storage
With the Printedly Online Designer, your online catalogue is fully hosted on our servers. Regardless of the WordPress hosting service you use, your Printedly online store includes:

* unlimited storage for your clip-art
* unlimited storage for customer uploads
* regular backups
* software upgrades and security updates

= Powerful Admin Panel
The Admin Panel allows you to upload your own clip-art, add your products and categories. Quote requests are shown in the orders section together with the design contents.

= Quick Quote Tool
The Quick Quote Tool allows customers to place a custom request without having to use the online designer. You may enter specific sections per product, for example, left chest, center back etc.

= Online Designer
Your customers can select the product they want, upload their own images, add their own custom text or use images from your clip-art panel.

== Installation ==


= Automatic installation (the easiest way) =

1. In your Wordpress admin backend, navigate to *Plugins → Add New*
2. Under Search, type in `Printedly` and click *Search*
3. In the search results find the *Printedly Online T-Shirt Designer* plugin and click *Install now* to install it
4. When plugin is installed click *Activate Plugin* link
5. Navigate to *Printedly → Products* page in your Wordpress admin backend and follow the instruction provided there
6. Add your own Product Catalog
7. Add the short code [printedly-designer] to your page for the designer, or [printedly-quote] for the quick quote tool.

= Alternative ways =

1. Download printedly plugin from this page (click *Download* button)
2. Unpack the downloaded zip archive
3. Upload printedly directory from the archive to the `/wp-content/plugins/` directory on your server
4. Activate the plugin through the 'Plugins' menu in WordPress
5. Add the short code [printedly-designer] to your page for the designer, or [printedly-quote] for the quick quote tool.

Please refer to this article for the details and troubleshooting on plugin installation in Wordpress:
http://codex.wordpress.org/Managing_Plugins#Installing_Plugins
